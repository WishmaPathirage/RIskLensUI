# RiskLens: AI Privacy Risk Analysis System

A modern, responsive frontend web application for analyzing privacy risks in AI inputs and documents.

## Project Structure
- **frontend/**: React + Vite web application source code.
- **backend/**: Placeholder for backend API services.
- **mlmodel/**: Placeholder for Machine Learning model files.

## Getting Started

The main application code is located in the `frontend` directory.

### Prerequisites
- Node.js (v18 or higher recommended)
- npm

### Installation & Running

1. **Navigate to the frontend directory**:
   You must move into the `frontend` folder where the React app lives.
   ```bash
   cd frontend
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. **View the setup**:
   Open your browser and navigate to `http://localhost:5173` (or the URL shown in your terminal).

## Features
- **Privacy Risk Scan**: Analyze text and files for PII and sensitivities.
- **Dashboard**: Real-time stats and recent activity.
- **Reports**: Detailed breakdown of risk analysis.
- **Secure Authentication**: JWT-based login/register flow.
